MANUAL DE USUARIO

Descripción. Este sistema de conteo vehicular, cuenta con un módulo de comunicación serial para conectarse a distintos tipos de sensores, posibilitando la detección automática de los carros y presentando una interfaz amigable, además cuenta con controles que permiten al usuario registrar manualmente ingresos o egresos vehiculares, al mismo tiempo éste sistema almacena la información de su estacionamiento, facilitando así una mejor gestión o análisis.

Tecnología. El sistema de conteo vehicular propuesto, es una aplicación de escritorio open source pensada para el sistema operativo Windows y publicada en 2022. Su desarrollo fue llevado a cabo a través de .NET Framework utilizando el lenguaje C#.

Requerimientos. Para ejecutar el sistema de conteo usted requiere de un equipo de cómputo con sistema operativo Windows 11 o 10, almenos 4GB RAM y 8GB de espacio en memoria de almacenamiento.

Sensores. El Sistema de conteo vehicular cuenta con un módulo de comunicación a través de un puerto serie, por lo que cualquier sensor programado para enviar un mensaje por dicho método de comunicación puede registrar ingresos, y en caso de egresos puede descontar si este mensaje de ingreso es repetido, esto puede funcionar en lectores de QR, Lectores RFID, Sistema de segmentación de placa, entre otros.
